# 🧪 Labs

This folder contains writeups for each hands-on lab room in the TryHackMe AI Security learning path.

## Structure

| Folder | Room | Difficulty | Status |
|--------|------|-----------|--------|
| [01-intro-to-ai-security](./01-intro-to-ai-security/) | Intro to AI Security | Easy | ✅ Complete |
| [02-adversarial-ml](./02-adversarial-ml/) | Adversarial ML | Medium | 🔄 In Progress |
| [03-prompt-injection](./03-prompt-injection/) | Prompt Injection | Medium | 🔄 In Progress |

## Writeup Convention

Each room folder contains a `writeup.md` file following this template:

```
## Room Overview
## Setup
## Tasks
### Task N — <Title>
#### Approach
#### Solution / Flag
## Key Takeaways
```

> **Note:** Flags are redacted or replaced with `FLAG{REDACTED}` to comply with TryHackMe's rules.
