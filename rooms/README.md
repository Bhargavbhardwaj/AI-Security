# 🏠 Room Writeups

This folder contains writeups for each TryHackMe room completed as part of the AI Security learning path.

## Writeup Template

Each writeup follows this structure:

```
## Room: <Room Name>
**Difficulty:** Easy / Medium / Hard
**Date Completed:** YYYY-MM-DD
**Link:** https://tryhackme.com/room/<room-name>

### Objective
Brief description of what the room covers.

### Tools Used
- Tool 1
- Tool 2

### Approach
Step-by-step walkthrough of how the room was completed.

### Key Takeaways
- Lesson 1
- Lesson 2
```

---

Writeups are added after completing each room. They serve as personal reference material and a portfolio of practical skills.
